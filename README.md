# HC_SIMULA — Guia: Exame Físico Pulmonar e Cardiovascular

Guia interativo de "como fazer" para exame físico do tórax (pulmonar e cardiovascular),
com imagens e áudios de ausculta (bulhas, galopes e sopros).

## Estrutura
- `index.html` — página principal (~67 kB)
- `img/` — 19 imagens (figuras e diagramas)
- `audio/` — 11 áudios de ausculta cardíaca
- `.nojekyll` — evita que o GitHub Pages processe o site com Jekyll

## Publicação no GitHub Pages
1. Envie **todos** os itens acima para o repositório `HC_SIMULA`, mantendo as pastas `img/` e `audio/`.
2. Em *Settings → Pages*, defina a branch (ex.: `main`) e a pasta raiz (`/root`).
3. Acesse: `https://wildsonbrandao-droid.github.io/HC_SIMULA/`

> Como o `index.html` agora é pequeno e os recursos estão em pastas, o upload pela
> interface web do GitHub não falha mais. Se preferir, arraste a pasta inteira de uma vez.
